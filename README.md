Exclusão de produtos com confirmação

Layout lado a lado (cadastrar + gerenciar)

🚀 Como Rodar o Projeto
Clone o repositório:

git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
Configure o Supabase:

Crie uma conta em supabase.com
Crie um projeto e uma tabela produtos com as colunas: nome, preco, imagem_url, Categoria
Copie a URL e a chave anon do projeto
Atualize as credenciais:

Substitua supabaseUrl e supabaseKey nos arquivos 
index.html
, 
login.html
 e 
admin.html
Abra o 
index.html
 no navegador e pronto! 🎉
